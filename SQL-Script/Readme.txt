Trước tiên, mở thư mục ShoeStore, rồi mở PowerShell và chạy lệnh:

dotnet ef database drop --force

Tiếp theo

dotnet watch

Sau đó quay lại thư mục này, chay file sau (chỉ cần nháy đúp), đợi hiện chữ Press Any Key to continue... rồi ấn phím bất kì để thoát

create_database.cmd
